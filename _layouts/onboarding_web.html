Web Integration

This integration guide provides an overview of the steps required to integrate your web property. For more details, download the full integration guide.

STEP 1 - Add JavaScript to Site

Insert the JavaScript tag below to all pages where native ad units will be served. 

<script type="text/javascript" src="//s.ntv.io/serve/load.js" async></script>

STEP 2 - Designate Content Landing Page URL

Publishers designate a URL and host a dedicated page where sponsored content will render. The page can reside under any URL path as long as it is unique and accessible.

Users that click on the Native Ad unit will remain on the original publication and be directed to that Content Landing Page.

STEP 3 - Create Content Landing Page Template

Add JS and meta tags to the content landing page. The recommended location is just before the closing </head> tag. 

<script type="text/javascript" src="//s.ntv.io/serve/load.js" async></script>
<meta http-equiv="X-UA-Compatible" content="IE=10" />
<meta name="robots" content="noindex, nofollow" />
