Advertisers

The Advertiser Service returns a list of advertisers by id and name. 

GET
https://api.nativo.net/v2/advertisers

Sample Request

{% include language_selector.html %} 
curl -H "nativo-token: $nativo_token" \
    -H "nativo-timestamp: $nativo_timestamp" \
    -H "nativo-hash: $nativo_hash" \
    -i https://api.nativo.net/v2/advertisers
import nativo

nativo.api_key = 'API_KEY'
nativo.api_secret = 'API_SECRET'

advertisers = nativo.Advertiser.list()
