Campaigns

The Nativo Campaigns Service returns a list of campaigns by id, name, and the id to the advertiser. 

GET
https://api.nativo.net/v2/campagins

Sample Request

{% include language_selector.html %} 
curl -H "nativo-token: $nativo_token" \
    -H "nativo-timestamp: $nativo_timestamp" \
    -H "nativo-hash: $nativo_hash" \
    -i https://api.nativo.net/v2/campaigns
import nativo

nativo.api_key = 'API_KEY'
nativo.api_secret = 'API_SECRET'

campaigns = nativo.Campaign.list()
