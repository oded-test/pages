Ad Server API

Nativo exposes endpoints that can be used for retrieving a list of articles or individual article based on a custom ﬁlter criteria that’s passed through Nativo’s ad call within the querystring.

Formats

The API allows retrieving data in sevral different formats, by using different extension when accessing the relevant endpoint:

  • Json: Using .json extension, will return the content in a Json format.
  • JavaScript: Using the .js extension, will return the content in a javascript format that can be executed on a page.
  • RSS: Using .rss extension, will return the content in an RSS format.

Note: To initiate the API-based ad call using the Javscript format, include the below code within the <head> section of the page. The JavaScript response will include a call to a callback function you can register to in order to retrieve the relevant data 

	        <head>
	<script type="text/javascript">
		// implement the callback function to retrieve the article(s)
	    function getContent() { 
	        if (data.articles.length > 0) {
	            // add your custom code here
	        }
	    }
	    // hook to the callback function to retrieve the article(s)
	    var Nativo = window.top.Nativo = window.top.Nativo || {}
	    Nativo.onArticlesReady = getContent;
	</script>
	<script src="http://jadserve.postrelease.com/..."></script>
</head>
